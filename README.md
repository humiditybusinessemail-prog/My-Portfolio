# Azzam Rizqi Azry Portfolio

Portfolio website for Azzam Rizqi Azry, a Web Developer & Engineer focused on building fast, scalable, and modern digital experiences for local businesses in Indonesia.

## About Me

- Nama: Azzam Rizqi Azry
- Profesi: Web Developer & Engineer
- Fokus: Website UMKM, company profiles, web engineering, performa, SEO, dan UX
- Goal: Reach over 50 clients with a strong local portfolio and professional web solutions.

## Portfolio Overview

This portfolio site uses a static HTML, CSS, and JavaScript setup with an interactive terminal-style entrance, service cards, and scroll-triggered sequencing.

### Features

- Terminal mockup welcome experience
- Clickable service cards with confirmation flow
- Scroll-progress animation
- GitHub-style contribution section
- Project showcase and contact CTA
- Deployment sandboxed to a `dist` folder for Netlify / GitHub Pages

## Run Locally

```bash
npm install
npm run dev
```

Open http://localhost:3000 and the portfolio will launch.

## Deploy

### Netlify

This project includes `netlify.toml` so you can connect the repository to Netlify and deploy directly from the root.

### GitHub Pages

A GitHub Actions workflow is included to deploy the static site to GitHub Pages automatically when pushing to `main`.
